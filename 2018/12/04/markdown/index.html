Markdown语法

简介

Markdown是一种轻量级的标记语言，以纯文本编写，以HTML格式发布。

标题

用1~6个”#”表示1到6级标题

段落

段落的前后要有空行，所谓的空行是指没有文字内容。
若想在段内强制换行的方式是使用两个以上空格加上回车（引用中换行省略回车）。

区块引用

在段落的每行或者只在第一行使用符号>,还可使用多个嵌套引用，如：

区块引用

嵌套

代码区块

代码区块的建立是在每行加上4个空格或者一个制表符（如同写代码一样）。
注意:需要和普通段落之间存在空行。

void main(){
    printf("HelloMarkdown")
}

或者用```language ```表示代码区块，这种方法还会有相应的语法高亮

强调

使用”*“或”“
一个表示斜体 相当于
两个表示_粗体 相当于
两个”~~“添加删除线

列表

有序列表

普通的用1.,2.,3.即可。
注意：”.”后面要空一格

  1. html
  2. js
  3. css

无序列表

每一项前面加”*”,”+”,”-“之一即可。
注意：连续使用两个列表，中间要空两行，否则视为上一个列表的延续

  • mysql
  • redis
  • oracle

分割线

分割线使用三个或以上”*”

链接

链接可以由两种形式生成：行内式和参考式。

行内式

[百度一下，你就知道](https://www.baidu.com “链接标题”)

链接文本1

自动链接

http://www.baidu.com

图片

同链接，前面加一个”!”
![](/images/ace.jpg “ace”)
