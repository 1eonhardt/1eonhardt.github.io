学习线性表的笔记

定义

具有相同数据类型的$n(n \geq 0)$个数据元素的有限序列。其一般表示为$L=(a1,a_2,\dots,a_i,a{i+1},\dots,a_n)$。除了表头元素$a_1$以外，每个元素有且仅有一个直接前驱。除表尾元素外，每个元素有且仅有一个直接后继。

特点：

  • 表中元素个数有限
  • 元素具有逻辑上的顺序
  • 元素都是数据元素？
  • 元素数据类型相同，意味着每个元素占有相同大小的存储空间
  • 元素具有抽象性，只关心元素之间的关系，而不关心具体存储的内容

主要操作：

  • 初始化
  • 求表长
  • 按位查找
  • 按值查找
  • 插入
  • 删除
  • 判空
  • 销毁
  • 输出

顺序表

即线性表的顺序存储。

基本操作实现

  • 插入：注意插入位置是否合法以及插入位置之后所有元素都要右移。
    • 最好情况：表尾插入 O(1)
    • 最坏情况：表头插入 O(n)
    • 平均情况：$\sum_{i=1}^{n+1}P_i(n-i+1)=\frac n2$
  • 删除：复杂度基本同上，一般要返回被删除的值
  • 按值查找：复杂度基本同上，注意考虑查找失败的情况
