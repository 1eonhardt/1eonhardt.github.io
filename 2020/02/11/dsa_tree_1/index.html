树学习笔记

相关术语

  • 节点(Node)：树的基本组成部分，每个节点具有名称，并且可以保存额外的数据项。
  • 度数(Degree): 一个节点的子节点的个数称为该节点的度。
  • 边(Edge)： 树的另一个基本组成部分
    • 每条边恰好连接两个节点，表示节点之间的关联
    • 每个节点都有一条入边(除根节点)和多条出边(除叶节点)。
  • 路径(Path)：由边连接在一起的有序节点列表
  • 子树(subtree)：一个节点及其所有子孙节点和相关边的集合
  • 层级(level)：从根节点到达一个节点路径包含的边的数量
  • 高度：树的层级的最大值。根节点所在层数为1。
  • 森林：森林是m棵互不相交的树的集合。

定义及特点

树是n个节点的有限集合，n=0时，称为空树。在任意一棵非空树中应满足：

  • 有且只有一个根节点
  • 除根节点外，每个节点及其子孙节点又构成一棵树，即子树。

性质

  1. 树的结点树等于所有节点的度数加1
  2. 度为m的树中第i层至多有mi−1m^{i-1}mi−1个节点
  3. 高度为h的m叉树至多有(mh−1)/(m−1)(m^h-1)/(m-1)(mh−1)/(m−1)个节点
  4. 具有n个节点的m叉树的最小高度为[logm(n(m−1)+1)][log_m(n(m-1)+1)][logm​(n(m−1)+1)]

二叉树

每个节点最多只有2个子节点，并且有左右之分，其次序不能任意颠倒。

和度为2的有序树的区别

  • 度为2的有序树至少有3个节点，而二叉树可以为空。
  • 度为2的有序树的次序是相对另一个子节点而言的，只有一个子节点时，就没有左右次序。而二叉树无论子节点是否为2，均需要确定其左右次序。

特殊的二叉树

满二叉树

高为h，且含有2h−12^h-12h−1个节点的二叉树，即每层都含有最多节点的树。

完全二叉树

每个节点都和满二叉树中编号1~n的节点一一对应的树。

特点：一个有n个节点的二叉树

  • 若i≤⌊n/2⌋i \le \lfloor n/2 \rfloori≤⌊n/2⌋，则为分支节点，否则为叶节点
  • 叶节点最多出现在底层和次底层
  • 每一个节点都有两个子节点(最多有一个例外，即最后一个叶子节点的父节点)
  • 只有一个左子节点的分支节点后面均为叶子节点
  • 若n为偶数，则最后一个节点是左子节点，其父节点只有一个子节点。若n为奇数，则每个分支节点都有左右子节点。

二叉排序树(BST)

比父节点小的key都出现在左子树，比父节点大的key都出现在右子树。
注意：插入顺序不同，生成的BST也不同

AVL树 – 平衡的二叉查找树

AVL是发明者名字缩写。

  • AVL树的每个节点都维护一个 平衡因子(balanceFactor = 左树高度 - 右树高度)。平衡因子大于0，称为左重，反之右重，为0则称平衡。
  • 如果每个节点的平衡因子都在 -1,0,1 之间(即左右子树高度之差不超过1)，则将该二叉查找树称为 平衡树。

二叉树的性质

  1. 非空二叉树的叶子节点数等于度数为2的节点数加1。
  2. 非空二叉树第k层至多有2k−12^{k-1}2k−1个节点
  3. 高为h的二叉树至多有2h−12^{h}-12h−1个节点
  4. 如果某个节点下标为p，则其左子节点为2p，右子节点为2p+1，父节点为p//2。(不能超过树的总节点数)
  5. 节点p所在深度为⌊log2p⌋+1\lfloor log_2p \rfloor+1⌊log2​p⌋+1

设度为0，1，2的节点数为n0n_0n0​，n1n_1n1​，n2n_2n2​。则有n=n0+n1+n2n=n_0+n_1+n_2n=n0​+n1​+n2​。由上面二叉树的图可知，除了根节点，树的每一条边对对应一个节点，节点的度数即为边数，故有n=n1+2n2+1n=n_1+2n_2+1n=n1​+2n2​+1。结合两式可得：n0=n2+1n_0=n_2+1n0​=n2​+1

二叉树的存储结构

顺序存储

根据二叉树的性质，完全二叉树和满二叉树采用顺序存储比较好。用数组下标确定节点在二叉树中的位置(因为数组下标是从0开始的，处理起来不方便，所以可以让数组第一个位置空着，这样数组下标和元素序数就对应了)。
